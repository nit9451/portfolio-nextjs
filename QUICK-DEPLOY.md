## Step 1: Create GitHub Repository (2 minutes)

1. **Go to**: https://github.com/new
2. **Repository name**: `portfolio-nextjs`
3. **Description**: `Fullstack Developer Portfolio - Nitish Kumar`
4. **Visibility**: Choose Public or Private
5. **IMPORTANT**: Do NOT check "Add a README file", "Add .gitignore", or "Choose a license"
6. **Click**: "Create repository"

## Step 2: Run the Push Script

Open PowerShell in this folder and run:

```powershell
.\push-to-github.ps1
```

The script will ask for:
- Your GitHub username
- Repository name (default: portfolio-nextjs)

Then it will automatically push everything!

## Step 3: Deploy to Vercel (Free Hosting - Optional)

1. Go to: https://vercel.com/new
